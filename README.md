# SCIM server sample
This is a simple sample of a SCIM server written using NodeJS and MongoDB as the data storage.

## Functionality supported

#### ResourceType: Users
CRUD operations
1. Create
2. Read
3. Update
4. Delete
5. Get by ID

#### ResourceType: Groups
CRUD operations
1. Create
2. Read
3. Update
4. Delete
5. Get by ID
